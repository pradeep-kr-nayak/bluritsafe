# BlurItSafe Privacy Policy

Last updated: July 14, 2026

BlurItSafe is a browser extension from GrazyTools.com that helps users blur sensitive content on a webpage before taking screenshots.

## What BlurItSafe does

BlurItSafe lets you:

- click elements on a webpage to blur or unblur them
- adjust blur intensity
- capture the visible page
- capture a selected area of the visible page

## What data BlurItSafe accesses

To provide these features, BlurItSafe may access:

- the currently active webpage, only after the user interacts with the extension
- visual content on the current tab needed to create a screenshot chosen by the user
- basic active-tab information needed to identify the current page during use

## How data is used

BlurItSafe uses page access only to:

- let the user select elements to blur on the current page
- apply the chosen blur effect locally in the browser
- generate a screenshot requested by the user
- download the screenshot to the user's device

## Data collection and sharing

- BlurItSafe does not sell user data.
- BlurItSafe does not share user data with third parties.
- BlurItSafe does not use page content or screenshots for advertising.
- BlurItSafe does not upload webpage content or screenshots to external servers as part of its core functionality.

## Local processing

BlurItSafe is designed so that blur operations and screenshot preparation happen locally in the user's browser session.

## Downloads

When a user chooses to save a screenshot, the generated image is downloaded to the user's device using the browser's download functionality.

## Permissions

BlurItSafe requests only the permissions needed for its current features:

- `activeTab` to work with the page the user is actively using after clicking the extension
- `scripting` to run the blur and selection logic on the active page
- `downloads` to save screenshots chosen by the user

## Changes to this policy

If BlurItSafe changes how it handles user data, this policy will be updated before or when those changes are released.

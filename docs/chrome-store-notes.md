# Chrome Web Store Notes

## Suggested single-purpose description

BlurItSafe helps users blur sensitive content on the current webpage and download privacy-safe screenshots.

## Suggested permission justifications

- `activeTab`
  Needed so the extension can work only on the page the user is actively viewing after clicking the extension action.

- `scripting`
  Needed to inject the blur-selection and area-selection logic into the active tab when the user starts using a feature.

- `downloads`
  Needed to save user-requested screenshots to the local device.

## Suggested store short description

Blur sensitive content on webpages and capture clean screenshots in one click.

## Suggested store long description

BlurItSafe helps you hide sensitive content before sharing screenshots.

Use it to click and blur text, images, tables, and sections on the current webpage, adjust blur intensity, then save either the full visible page or a selected area.

BlurItSafe is designed for privacy-safe sharing of webpages, dashboards, chats, documents, and other browser content.

Core features:

- Click any supported element to blur or unblur it
- Blur text, images, tables, and larger page sections
- Adjust blur intensity before capture
- Capture the full visible page
- Capture a selected area only
- Save screenshots directly to your device

## Suggested support / reviewer note

This extension does not require an account, external login, or paid subscription to test all user-facing features.
